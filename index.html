What is the cube root?
The cube root of a number x is a number y such that y^3 = x all nonzero real numbers have exactly on ereal cube root and a pair of complex conjugate cube roots.
How to calculate cube root?
To calculate cube root you should use the following formula:
(x)^1/3
Example: Find the cube root of the number 8?
Using the cube root formula the answer will be:
(8)^1/3 = 2
Why use cube root?
The cube root is usually used to solve cubic equations.
